Bootstrap
===================

This repository contains the files from the [course](https://cursos.alura.com.br/course/bootstrap-criacao-single-page-responsiva)

> ##Classes:

> - `Class 1` : Introduction
> - `Class 2` : Starting the implementation following the layout
> - `Class 3` : Show all the projects from the Company
> - `Class 4` : Show all the customers review with carousel
> - `Class 5` : Contact form and Institutional Video from the Company
> - `Class 6` : Address on the Footer section
> - `Class 7` : Navigating through the sections with Navbar
> - `Class 8` : Featured Header
> - `Class 9` : Turning the site Responsive to work on Bigger Screens
